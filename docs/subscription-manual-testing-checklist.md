# Subscription Manual Testing Checklist

Use this checklist to manually validate subscription plan setup, tenant assignment, login/session enforcement, seat limits, feature gating, and cross-tenant safety.

## Test Setup

Prepare the following before executing the checklist:

- `SUPER_ADMIN` account
- `ADMIN` account that owns 1 tenant
- `ADMIN` account that owns 2 tenants
- `MANAGER`, `CREDIT_INVESTIGATOR`, `LOAN_OFFICER`, and `CASHIER` accounts for at least one tenant
- Tenant A on `BASIC`
- Tenant B on `PROFESSIONAL`
- Tenant C on `ENTERPRISE`
- Sample customers, loans, payments, activity logs, and branding assets in each tenant
- Subscription schema migrated with `plans`, `tenants.plan_id`, `tenants.plan_code`, `tenants.subscription_status`, and `tenants.subscription_expires_at`
